# Ally – Помощник при аллергии

**Ally** — это Android‑приложение, которое помогает контролировать аллергию, собирая всю важную информацию в одном месте. Ведите медицинский паспорт (аллергены, лекарства, контакты), записывайте триггеры и отслеживайте симптомы с интенсивностью — все данные хранятся локально на вашем устройстве.

---
## 📱 Функционал

### 🧾 Медицинский паспорт (Passport)
- Добавление и отображение аллергенов
- Список принимаемых препаратов
- Контакты врача и родственника
- Режим редактирования (переключение "Изменить / Готово")
- Автоматическое сохранение данных

---

### 📓 Дневник (Diary)
- Добавление записей о:
  - предшествующих событиях
  - принятых лекарствах
- Объединение данных в одну запись
- Отображение истории записей
- Форматированное время записи
- Новые записи отображаются сверху

---

### ⚠️ Симптомы (Symptoms)
- Список типовых симптомов:
  - чихание, зуд, кашель и др.
- Выбор симптома
- Указание интенсивности через слайдер:
  - слабая / средняя / сильная
- Автоматическое сохранение
- История симптомов с:
  - временем
  - интенсивностью
  - временем суток

## 📋 Требования
- Android Studio (последняя стабильная версия)
- JDK 11 или выше
- Android SDK с API 24+ (минимальная поддерживаемая версия)


## 🛠️ Сборка и запуск

1. Клонирование проекта

   ```bash
   git clone https://github.com/manwoodt/ally.git
   cd ally
   ```

2. Откройте проект в Android Studio.
   - Запустите Android Studio
   - Выберите Open an existing project и укажите папку ally.

3. Соберите проект
   Gradle автоматически синхронизирует зависимости. При необходимости выполните File → Sync Project with Gradle Files.

4. Запустите приложение
   - Подключите Android-устройство (API 24+) или запустите эмулятор.

   - Нажмите кнопку Run (зелёный треугольник) или используйте сочетание клавиш Shift+F10.
   
   - Выберите целевое устройство. Приложение установится и запустится автоматически.
