# Ally – Помощник при аллергии

**Ally** — это Android‑приложение, которое помогает контролировать аллергию, собирая всю важную информацию в одном месте. Ведите медицинский паспорт (аллергены, лекарства, контакты), записывайте триггеры и отслеживайте симптомы с интенсивностью — все данные хранятся локально на вашем устройстве.

---

## 📱 Возможности

- **Медицинский паспорт**  
  – Список аллергенов, лекарств (название + дозировка), имя врача и контакт родственника.  
  – Режим редактирования: добавляйте и удаляйте информацию на лету.

- **Дневник триггеров**  
  – Фиксируйте съеденные продукты или другие триггеры с временной меткой.  
  – Записи отображаются в обратном хронологическом порядке.

- **Трекер симптомов**  
  – Быстрый выбор распространённых симптомов (чихание, зуд, слёзы, кашель).  
  – Выбор интенсивности (слабая, средняя, сильная) с автоматическим определением времени суток (утро, день, вечер).  
  – История всех записанных симптомов.

- **Локальное хранение**  
  – Все данные сохраняются через Jetpack DataStore с использованием Kotlinx Serialization.  
  – Не требуется интернет — ваши данные остаются приватными.

- **Современный интерфейс**  
  – Построен на Jetpack Compose и Material 3 в пастельных тонах.  
  – Нижняя навигация для быстрого переключения между разделами.
